Define links



  1. In this screen you can modify the graph according to selected option.
  2. When Make link option is selected, a link between the next two nodes you click will be made.
  3. When Remove link option is selected, the next link you select will disappear.
  4. At any time you can drag any node, but if a make/remove link option is selected, then it may create links. To just drag nodes the Just drag options should be selected.
  5. You also can set the link in the text area by writing 1 link per line in the format:
    nodeA nodeB cost
    This is the recommended way to add links because you have control on the cost of the links, you can omit cost, then default cost would be 1.
  6. After you finished the graph construction, click next.
    The links in the text area will be built after you click the next button.
